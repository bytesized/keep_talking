New Bomb

This tool remembers some details about the bomb when they are provided. Details about the bomb's serial number, for example, and how many batteries it has do not change over the course of a single bomb. The "Reset Bomb" button, above, causes those details to be forgotten, readying this tool to start a new bomb.
